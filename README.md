ٍAegis API Testing Suite

Aegis API Testing Suite is a comprehensive API testing project designed to validate RESTful services using Postman with automated execution through Newman and CI integration via Jenkins.

The project demonstrates real-world API testing workflows including request validation, automated test scripting, environment management, authentication handling, negative testing, and data-driven execution. It reflects industry-standard QA practices for ensuring API reliability, performance, and correctness.

Key Capabilities

Testing REST APIs using GET, POST, PUT, PATCH, and DELETE methods

Automated validation of status codes, response body, headers, and response time

JavaScript-based test assertions

Environment variables management

JWT authentication testing

Negative testing scenarios

Data-driven testing using JSON and CSV files

Command-line execution using Newman

Continuous Integration pipeline with Jenkins

Automated HTML test report generation

Objective

To showcase practical API testing skills and automation techniques used in modern software quality assurance and CI/CD environments.
